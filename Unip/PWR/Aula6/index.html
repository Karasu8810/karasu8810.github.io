Aula 06 - CSS e HTML

Aprendemos a fazer as bordas de um site e as bordas de apenas um referencial
Para montarmos um espaçamento das bordas usamos o "padding" que pode definir essa distância:

  • Adicionar a tag referencial no CSS:
    body{}
  • Atribuir os devidos espaçamentos:
    body{
    padding-left:10px;
    padding-top:10px;
    }
  • Aplicar o arquivo CSS no HTML:
    "link rel="stylesheet" href="style.css""

Na mesma aula aprendemosw a utilizar o "border", uma ferramenta que gera um contorno na tag onde pode-se referenciar cor e tamanho:
  • Adicionar a tag referencial no CSS:
    body{}
  • Atribuir a espessura das bordas:
    body{
    border:10px;
    }
  • Aplicar uma cor para as bordas:
    body{
    border:10px solid blue;
    }
    solid: utilizado sempre para quando se refere a uma cor no CSS
  • Aplicar o arquivo CSS no HTML:
    "link rel="stylesheet" href="style.css""
